# SF_v12_TVMode

Main mode is still the 35-minute final build.

A tiny **TV** button appears in the bottom-right corner. Click it, enter password `<set TV_CODE>`, and the page reloads into TV test mode.

In TV mode, when the admin starts the lobby using admin code `<set ADMIN_CODE>`, the match runs for 5 minutes with 20-second respawn.

Click **TV** and confirm to return to normal 35-minute final mode.

Important: this is one shared server arena. Do not start TV mode on the same deployed server while a real final match is already running.
