# SF_v14_DrifterShips

- Removed the old neutral-ship wording from the player-facing build.
- Neutral filler ships are now called drifter ships.
- Drifter ships appear from Phase 1 onward so the arena feels less empty.
- Drifter ships are kept smaller than or roughly equal to starting player ships.
- Drifter ship rewards are lighter so they fill the map without inflating player size too fast.
- Existing final settings remain: 35-minute main run, 5-minute TV mode, 5-minute inactivity timeout.
